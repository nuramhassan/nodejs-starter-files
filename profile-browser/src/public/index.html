Welcome to [Your Name]'s Profile RESTful API

This page contains some simple ways to get information from a profile file.

  • GET/

    Returns the index.html (this page).

  • GET/api/profiles

    Returns a list of reduced profiles information from the profiles.json file. The expected results should only return each profiles id, firstname and lastname

    Response Body:

    [
    { "id":1, "firstname" : "Maya", "lastname" : "Angelou", },
    { "id":2, "firstname" : "Toni", "lastname" : "Morrison", }
    ....
    ....
    ]

  • GET/api/profiles/:id

    Returns a single profile based on a querystring parameter entered by the client. The API should search the profiles in the list and return the firstname and lastname of the profile that matches, e.g.

    Request URL:

    /api/profiles/2

    Response Body:

    [
    { "id":2, "firstname" : "Toni", "lastname" : "Morrison", }
    ]

  • POST/api/profiles

    Post a new profile object to server. Will return status 200 and success if successful. Will return Status 500 fail if unsuccessful. Profile must be valid JSON object, e.g:

    Request Body:

    [
    { "firstname" : "Jessica", "lastname" : "Jones", "bio": "Detective" }
    ]

    Response:

    Status Code: 200 OK
    Body: Success

    N.B. To test this call you will need an API tester such as Postman

  • PUT/api/profiles/:id

    Replaces an existing profile object. Will return status 200 and success if successful. Returns 404 Not Found if the id doesn't exist. Will return Status 500 error if unsuccessful. Profile must be valid JSON object, e.g.:

    Request URL:

    /api/profiles/5

    Request Body:

    [
    { "firstname" : "Jessica Campbell", "lastname" : "Jones Cage", "bio": "Detective" }
    ]

    Response:

    Status Code: 200 OK
    Body: Success

    N.B. To test this call you will need an API tester such as Postman

  • PATCH/api/profiles/:id

    Updates an existing profile object. Will return status 200 and success if successful. Returns 404 Not Found if the id doesn't exist. Will return Status 500 fail if unsuccessful. Ignores new object attributes. In the example below the height attribute would be ignored, e.g:

    Request URL:

    /api/profiles/5

    Request Body:

    [
    { "firstname" : "Jessica Campbell", "lastname" : "Jones Cage", "bio": "Detective", "height": 5.7 }
    ]

    Response:

    Status Code: 200 OK
    Body: Success

    N.B. To test this call you will need an API tester such as Postman

  • DELETE/api/profiles/:id

    Delete profile by id. Returns 404 Not Found if the id doesn't exist.

    Request URL:

    /api/profiles/5

    Response:

    Status Code: 200 OK
    Body: Success

    N.B. To test this call you will need an API tester such as Postman
